Project One: Calculator

Project Two ❯

You now know enough to program a basic calculator! This project makes use of Python’s basic arithmetic operators as well as Python’s math module to perform the following calculations.

  • Addition.
  • Subtraction.
  • Multiplication.
  • Division.
  • Modulo.
  • Absolute value.
  • Square root.
  • Cube root.
  • Exponent.
  • Sine (both degrees and radians).
  • Cosine (both degrees and radians).
  • Tangent (both degrees and radians).

It should also support using the inputs “pi” and “e” to perform operations with π and e, respectively, as well as the ability to enter random expressions in the form “rand(#,#)” where “#” represents any number. Note that there’s no space after the comma! This makes it easier to parse. The term parse means to pull information from something and process it. In this project, you’ll be parsing the user’s string input to pull out the important pieces of information to figure out what they want to calculate.
