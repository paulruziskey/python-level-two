  • Create a function called rand_nums which takes in a size, a lower limit, and an upper limit, and returns a set of random integers between the lower limit and the upper limit with a length equal to the size. Use this function to generate two sets of five random integers between one and 10, then print the sets, the numbers they have in common, the numbers unique to one set, and the numbers that were generated (i.e., all the numbers between both sets)
