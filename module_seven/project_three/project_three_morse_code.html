Project Three: Morse Code

For this project, you’ll use a dictionary to translate a message into Morse code! The following table shows the Morse code for each letter in the Latin alphabet.

B -...

S ...
T -

V ...-
W .--
X -..-
Y -.--

You should first make a dictionary which maps the Latin letters to their Morse-code equivalents. You should then write a function called to_morse which accepts an English message and returns the message translated into Morse code. You should then ask the user to enter a message. Finally, output the user’s message translated into Morse code! Keep in mind that the case of letters doesn’t matter, and punctuation should be ignored. Example output is shown below.
